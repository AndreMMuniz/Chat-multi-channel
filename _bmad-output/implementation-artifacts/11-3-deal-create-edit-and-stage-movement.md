# Story 11.3: Deal Create/Edit and Stage Movement

**Status:** ready-for-dev  
**Epic:** 11 — Projects / Pipeline Kanban  
**Story Points:** 5  
**Priority:** High  
**Created:** 2026-05-02

---

## User Story

**As an operator,** I want to create, edit, and move deals between stages so that the pipeline is operational rather than static.

---

## Scope

Add interactive deal management to the kanban workspace:

- create deal flow
- edit deal flow
- drag-and-drop stage movement
- modal or sheet for deal details

---

## Acceptance Criteria

- [ ] Users can create a new deal from the workspace
- [ ] Users can open an existing deal and edit its fields
- [ ] Users can move a deal between columns
- [ ] Drag-over and drop states are clearly visible
- [ ] The UI does not require leaving the `Projects` workspace to manage a deal
- [ ] English labels are maintained throughout

---

## Notes

- If delete behavior is included, keep it secondary and deliberate.
- Do not introduce CRM automation in this story.
